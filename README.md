# Angeltrainer Bayern — Privacy & Legal

GitHub-Pages-Site mit Datenschutzerklärung und Impressum der Angeltrainer Bayern-iOS-App.

Live: https://cemoso88.github.io/angelplan-privacy/

## Inhalt

- `index.html` — Privacy Policy / Datenschutzerklärung (DE/EN)
- `impressum.html` — Impressum / Legal Notice (DE)

## Deployment

Push to `main` → GitHub-Pages serviert automatisch aus dem Repo-Root.

App-Settings linken auf:
- `https://cemoso88.github.io/angelplan-privacy/` → Privacy
- `https://cemoso88.github.io/angelplan-privacy/impressum.html` → Impressum
